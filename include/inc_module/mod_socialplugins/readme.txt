Installation
------------

The installation procedure is quick and easy!

Unzip the downloaded file to your phpwcms installation.

To activate the module go to the modules section an click the new 'Social Plugins' link.

That's it!
The script adds 1 tables to the phpwcms database.


Update
------
Unzip the downloaded file to your phpwcms installation and overwrite the existing files.
Update all your existing Facebook, Twitter and Google+ buttons and widgets in the module section to add the new functions.

The update does not change the DB.
